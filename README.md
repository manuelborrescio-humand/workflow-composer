# Workflow Composer

**Equipo:** humand.after.all — Manuel, Cristiano, Martin, Hernan
**Evento:** Humand AI Hackathon 2026 | 18–19 marzo

---

## Que es Workflow Composer

Web app que convierte descripciones en lenguaje natural de procesos de HR en diagramas visuales de workflows, usando el formato HWL (Humand Workflow Language) real de la plataforma Humand.

## Stack

| Herramienta | Uso |
|---|---|
| **Next.js + TypeScript** | Framework |
| **Claude API** | Agente que genera el JSON HWL |
| **React Flow** | Diagrama visual |
| **Tailwind CSS** | Estilos |
| **Zod** | Validacion JSON |
| **Vercel** | Deploy |

## Setup local

```bash
npm install
cp .env.example .env.local
# Completar las keys en .env.local
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000).
